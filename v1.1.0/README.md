# Barble Agent v1.1.0

## Novità
- Supporto stampa logo su ricevute ed etichette (jimp)
- Supporto stampa etichette ESC/POS

## Installazione
1. Scarica il binario per il tuo sistema operativo
2. Copia `barble-config.example.json` come `barble-config.json` nella stessa cartella del binario
3. Configura `serverUrl`, `restaurantId` e `token` (scaricabile da Settings → Stampanti)
4. Avvia il binario

## File
- `barble-agent-linux` — Linux x64
- `barble-agent-macos` — macOS x64
- `barble-agent-win.exe` — Windows x64
- `barble-config.example.json` — Configurazione esempio
